# FCFS (First Come First Serve)

FCFS is the simplest CPU scheduling algorithm.

Processes are executed based on arrival order.

---

# Characteristics

- Non-preemptive
- Simple implementation
- Queue-based scheduling

---

# Advantages

- Easy to implement
- Low overhead

---

# Disadvantages

- Convoy Effect
- Poor average waiting time

---

# Real World Usage

- Printer queues
- Ticket systems
- Batch processing systems

---

# Time Complexity

O(n)
