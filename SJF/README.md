# Shortest Job First (SJF)

SJF selects the process with the smallest burst time.

---

# Characteristics

- Can be preemptive or non-preemptive
- Optimizes average waiting time

---

# Advantages

- Efficient CPU usage
- Low average turnaround time

---

# Disadvantages

- Starvation problem
- Difficult burst time prediction

---

# Real World Usage

- Operating systems
- Cloud task scheduling
- CPU optimization systems

---

# Time Complexity

O(n log n)
