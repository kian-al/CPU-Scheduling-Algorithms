# Round Robin Scheduling

Round Robin allocates a fixed time quantum to each process.

Processes execute in circular order.

---

# Characteristics

- Preemptive scheduling
- Fair CPU allocation

---

# Advantages

- Prevents starvation
- Good response time

---

# Disadvantages

- High context switching
- Quantum size is critical

---

# Real World Usage

- Modern operating systems
- Multi-user systems
- Interactive systems
- Network scheduling

---

# Time Complexity

O(n × cycles)
